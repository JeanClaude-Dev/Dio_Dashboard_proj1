# 🎮 Xbox Game Pass - Sales Performance Dashboard

## 📌 Sobre o Projeto
Este projeto foi desenvolvido para transformar dados brutos de assinaturas do Xbox Game Pass em insights estratégicos. O objetivo é fornecer uma visão clara sobre o faturamento, mix de produtos e comportamento de consumo dos usuários, auxiliando na tomada de decisão baseada em dados (Data-Driven).

O dashboard foca em métricas de receita (Revenue), adoção de serviços adicionais (Season Passes) e eficiência de campanhas de desconto.

---

## 🚀 Tecnologias Utilizadas
* **Microsoft Excel:** Motor principal para processamento e visualização.
* **Tabelas Dinâmicas:** Para sumarização rápida de grandes volumes de dados.
* **Campos Calculados:** Implementação de lógica de negócio direta na Pivot Table (ex: Ticket Médio).
* **Segmentação de Dados (Slicers):** Filtros interativos para análise dinâmica.
* **Dashboard Design:** Layout otimizado para UX/UI focada em Dashboards Corporativos.

---

## 📊 Perguntas de Negócio Respondidas
O dashboard foi estruturado para responder aos seguintes KPIs:
1.  **Faturamento Total:** Qual o volume bruto de vendas por período?
2.  **Market Share de Planos:** Qual a representatividade dos planos *Ultimate*, *Standard* e *Core* na receita total?
3.  **Performance de Add-ons:** Qual o impacto nas vendas com a inclusão do *EA Play* e *Minecraft Season Pass*?
4.  **Ticket Médio (ARPU):** Qual o gasto médio por usuário considerando descontos e cupons?
5.  **Ciclo de Retenção:** Qual a proporção de assinaturas anuais vs. mensais e o status de renovação automática.

---

## 📈 Insights Extraídos
* **Otimização de Receita:** O plano *Ultimate* possui o maior Ticket Médio devido à integração nativa com o EA Play.
* **Impacto de Promoções:** O uso de cupons de desconto foi identificado como um driver de conversão para o plano *Standard*.
* **Fidelização:** Usuários com renovação automática ativa apresentam um ciclo de vida (LTV) superior.

---

## 🛠️ Como Reproduzir
1.  Faça o download do arquivo `projeto_dashboard.xlsx`.
2.  Certifique-se de habilitar as macros/atualizações se solicitado.
3.  Navegue pelas **Segmentações de Dados** na lateral para filtrar por tipo de plano ou renovação.
4.  Para ver a lógica de cálculo, acesse a aba oculta/auxiliar de `Cálculos`.
